## Format

```
| YYYY-MM-DD | [topic-slug](YYYY-MM-DD-topic-slug.md) | in_progress / done / paused |
```

Status values:
- `in_progress` — actively being worked on
- `paused` — work started, currently on hold (note the reason in the session file)
- `done` — finished, ready to be moved to `_archived/` on next session start

See `.ai/README.md` for the file template and the 10-day archival rule.
